# 可爱风自动世界模拟游戏设计文档 v0.1

> 暂定方向：玩家先绘制一个可爱的动物文明世界，随后世界自动演化。玩家主要以观察者、导演、轻度干预者的身份，看国家发展、贸易、战争、分裂、和解与王朝更替。

---

## 1. 项目概念

### 1.1 一句话介绍

一款清新可爱风的单机自动世界模拟游戏：玩家提前画好地形、国家与种族分布，然后观察狗国、猫猫国、狐狸国、地鼠国、獾国等动物文明在世界中自动发展、外交、战斗、衰落与重生，并可以在关键时刻轻度干预世界走向。

### 1.2 核心体验

本游戏的核心不是传统策略游戏里的“玩家亲自管理每一步”，而是：

- **创造一个世界**
- **观察这个世界自己运转**
- **从自动演化中获得故事**
- **在想介入时进行轻量干预**

玩家更像是：

- 世界导演
- 动物文明观察员
- 童话史官
- 温和的箱庭神明

### 1.3 参考体验关键词

- WorldBox 式自动演化
- RimWorld / Dwarf Fortress 式故事涌现
- 文明模拟，但更轻、更可爱
- 单机箱庭
- 动物王国
- 清新、简洁、低压力
- 观察大于操作
- 世界自己讲故事

## 2. 设计原则

### 2.1 可爱外表，真实逻辑

画面可以非常可爱，但世界运转逻辑要自洽。

国家不是随机打架，而是因为：

- 资源不足
- 边境摩擦
- 贸易冲突
- 旧仇复燃
- 继承危机
- 贵族叛乱
- 宗教 / 传统 / 种族倾向
- 地形与战略价值

### 2.2 玩家不是操作员，而是观察者

玩家不需要每分钟点大量按钮。

理想状态是：

> 玩家布置世界后，游戏自己跑起来。玩家可以暂停、观察、查看历史、点击角色、轻度干预。

### 2.3 种族必须有行为味道

每个种族不只是数值不同，而是行为风格不同：

| 种族 | 行为味道 |
|---|---|
| 狗国 | 忠诚、秩序、军团、防线 |
| 猫猫国 | 独立、优雅、贵族、派系 |
| 狐狸国 | 外交、贸易、阴谋、代理人 |
| 地鼠国 | 地下、工程、矿脉、储备 |
| 獾国 | 防御、顽固、山堡、复仇 |

玩家应该能从行为上感受到“这很猫”“这很狐狸”“这很獾”。

### 2.4 历史日志是核心产物

本游戏最终产出的不是单一胜负，而是一个可分享的世界故事。

例如：

> 第27年春，狗国训练场升级，士气显著提升。  
> 第29年秋，地鼠国挖通新矿脉，引发獾国边境警戒。  
> 第31年冬，猫猫国与狐狸国达成贸易协定，但黑猫夜会暗中调查狐狸使节。  

### 2.5 离谱可以发生，但必须可解释

允许出现“地鼠国统一大陆”“狐狸国扶持三个傀儡王”“猫猫国因为毛线球节经济崩溃”等离谱故事。

但这些结果最好能被玩家追溯：

- 谁先占了资源点
- 谁签了盟约
- 谁暗杀了国王
- 谁修了地下通道
- 哪次饥荒改变了国运

## 3. 核心玩法循环

### 3.1 主循环

```text
绘制世界
  ↓
放置国家 / 种族 / 资源
  ↓
启动自动演化
  ↓
观察地图变化与事件日志
  ↓
查看国家 / 角色 / 历史
  ↓
轻度干预或继续观察
  ↓
世界进入新的阶段
```

### 3.2 玩家阶段

#### 阶段一：设计世界

玩家可以绘制：

- 草地
- 森林
- 山地
- 河流
- 湖泊
- 地下入口
- 国家边界
- 初始城市
- 资源点
- 种族聚落

#### 阶段二：观察演化

世界开始自动运行：

- 国家扩张
- 人口增长
- 资源采集
- 城镇发展
- 贸易路线形成
- 外交关系变化
- 小规模冲突
- 战争爆发
- 王朝更替
- 国家分裂或合并

#### 阶段三：轻度干预

玩家可以通过“神明干预”影响世界：

- 祝福某片土地
- 促进两个国家和谈
- 召唤英雄
- 引导迁徙
- 制造灾害
- 平衡过强势力
- 放置新资源
- 改变地形

## 4. 世界与地图系统

### 4.1 地图视角

推荐使用轻量俯视 / 斜俯视地图。

画面需要：

- 清新
- 简洁
- 可读性强
- 不是页游式密集 UI
- 不是过度复杂的插画海报
- 功能尽量“长在世界里”

### 4.2 基础地形

| 地形 | 作用 |
|---|---|
| 平原 | 农业、城市、骑兵适性高 |
| 森林 | 木材、隐蔽、猫 / 狐狸适性较高 |
| 山地 | 矿石、防御、獾国适性高 |
| 河流 | 农业、贸易、天然边界 |
| 湖泊 | 鱼类资源、景观、交通 |
| 沼泽 | 疾病、低开发、隐蔽 |
| 沙地 | 沙狐贸易、补给困难 |
| 雪地 | 北极狐适性高，农业弱 |
| 地下层 | 地鼠核心玩法，矿井、地道、避难所 |

### 4.3 资源

资源应尽量可爱化，而不是硬核中世纪资源表。

| 资源 | 用途 |
|---|---|
| 食物 | 人口增长、军队补给 |
| 木材 | 建筑、桥梁、工具 |
| 石材 | 城墙、城堡、山堡 |
| 铁矿 | 武器、护甲、工具 |
| 鱼 | 猫猫国偏好 |
| 骨头 | 狗国偏好 |
| 蜂蜜 | 獾国 / 狐狸国偏好 |
| 浆果 | 狐狸国偏好 |
| 蘑菇 | 地鼠国偏好 |
| 猫薄荷 | 猫猫国特殊资源 |
| 闪闪石 | 高级建筑、奇观、外交礼物 |

### 4.4 地图可读性要求

地图上最多同时强调：

- 国家边界
- 当前选中国家
- 正在发生的事件
- 主要道路 / 河流
- 重要资源点

不要同时堆叠过多 UI 图层，否则会产生页游感。

## 5. 五大种族与国家设定

## 5.1 狗国

### 定位

狗国是秩序型、王权型、军团型文明。

他们擅长：

- 组织
- 防御
- 军事纪律
- 守信
- 边境巡逻
- 联盟

### 核心机制：忠诚链条

狗国国民和官员基础忠诚较高。

优势：

- 稳定性高
- 不容易叛乱
- 军队纪律强
- 容易形成强王权

风险：

- 好国王会让国家强大
- 昏君也会被认真执行命令
- 国家可能“稳定地走向灾难”

### 亚种

| 亚种 | 定位 | 特性 |
|---|---|---|
| 牧羊犬族 | 官员 / 将军 | 行政效率高，军队纪律强 |
| 猎犬族 | 斥候 / 追击兵 | 侦察与边境巡逻强 |
| 金毛族 | 外交官 / 医者 | 民心、外交、恢复能力强 |
| 柴犬族 | 武士 / 地方贵族 | 自尊高，战斗强，服从性略低 |
| 哈士奇族 | 探险者 / 混乱变量 | 士气高，容易触发奇怪事件 |
| 杜宾族 | 近卫 / 军警 | 镇压叛乱、保护王室强 |
| 柯基族 | 宫廷贵族 | 魅力高，野心高，喜剧事件多 |

### 行为倾向

| 行为 | 倾向 |
|---|---|
| 结盟 | 高 |
| 遵守条约 | 很高 |
| 主动宣战 | 中 |
| 防御边境 | 很高 |
| 镇压叛乱 | 高 |
| 军事发展 | 高 |
| 商业发展 | 中低 |
| 接纳移民 | 中 |

---

## 5.2 猫猫国

### 定位

猫猫国是独立、优雅、贵族、文化、商业与派系政治文明。

他们擅长：

- 商业
- 文化
- 城市发展
- 贵族政治
- 刺客与情报
- 外交魅力

### 核心机制：派系自治

猫猫国不是铁板一块，而是多个强势派系共存：

- 王室
- 贵族家族
- 商会
- 学院
- 黑猫夜会
- 地方城市

优势：

- 文化强
- 商业强
- 城市繁荣
- 容易产生英雄和名人

风险：

- 内部派系斗争频繁
- 国王不一定管得住全国
- 高税收容易触发商会罢市
- 外交上容易“傲娇”

### 亚种

| 亚种 | 定位 | 特性 |
|---|---|---|
| 橘猫族 | 商人 / 富民 | 商业强，食物消耗高 |
| 黑猫族 | 刺客 / 秘术师 | 夜战、暗杀、情报强 |
| 布偶猫族 | 王室 / 贵族 | 魅力、外交、联姻强 |
| 暹罗猫族 | 学者 / 谋士 | 科技、文化、法律强 |
| 缅因猫族 | 重步兵 | 体型大，近战强 |
| 三花猫族 | 民间领袖 / 女王 | 民心高，英雄事件多 |
| 奶牛猫族 | 艺术家 / 混乱小贵族 | 创造力强，稳定性低 |

### 行为倾向

| 行为 | 倾向 |
|---|---|
| 商业发展 | 很高 |
| 文化发展 | 很高 |
| 间谍行动 | 高 |
| 内部派系斗争 | 很高 |
| 主动宣战 | 中低 |
| 遵守条约 | 中 |
| 镇压叛乱 | 中低 |
| 追求奢侈资源 | 很高 |

---

## 5.3 狐狸国 / 狐狸果

### 定位

狐狸国是外交、贸易、情报、阴谋和代理人战争文明。

他们擅长：

- 谈判
- 贸易
- 间谍
- 扶持叛乱
- 挑拨外交
- 复杂条约
- 控制贸易路线

### 核心机制：双面外交

狐狸国拥有多层外交关系：

- 表面关系
- 真实关系
- 秘密协议
- 代理势力
- 间谍渗透

例如：

> 狐狸国表面上与狗国结盟，暗中资助猫猫国贵族，又偷偷卖矿石给地鼠国。

### 亚种

| 亚种 | 定位 | 特性 |
|---|---|---|
| 赤狐族 | 外交官 / 贵族 | 平衡型，外交强 |
| 银狐族 | 王室 / 间谍头子 | 魅力、阴谋、宫廷影响强 |
| 北极狐族 | 寒地拓荒者 | 雪地生存强 |
| 沙狐族 | 商队 / 行会 | 沙漠贸易强 |
| 耳廓狐族 | 斥候 / 情报员 | 感知强，侦察强 |
| 藏狐族 | 冷面官员 / 反间谍 | 稳定，降低被渗透概率 |
| 九尾狐族 | 传奇单位 | 超自然、预言、魅惑，稀有事件 |

### 行为倾向

| 行为 | 倾向 |
|---|---|
| 结盟 | 很高 |
| 遵守条约 | 中低 |
| 主动宣战 | 低 |
| 代理人战争 | 很高 |
| 间谍行动 | 很高 |
| 商业发展 | 高 |
| 扶持叛乱 | 很高 |
| 战败求和 | 高 |

## 5.4 地鼠国

### 定位

地鼠国是地下、工程、矿业、储备与避战文明。

他们擅长：

- 挖矿
- 地下城
- 隧道
- 粮食储备
- 防御避难
- 工程建设
- 战略偷袭

### 核心机制：地下层

地鼠国可以开发地下地图：

- 地下城市
- 地道
- 矿井
- 粮仓
- 避难所
- 地下河
- 偷袭出口

其他国家看到的是地表边界，地鼠国真正的势力可能在地下。

### 亚种

| 亚种 | 定位 | 特性 |
|---|---|---|
| 普通地鼠族 | 农民 / 矿工 | 稳定生产，基础挖掘 |
| 鼹鼠族 | 工程师 / 地道兵 | 挖掘、偷袭、矿井强 |
| 仓鼠族 | 粮仓管理员 / 商人 | 储粮强，抗饥荒 |
| 土拨鼠族 | 哨兵 / 预警者 | 提前发现灾害和入侵 |
| 裸鼹鼠族 | 地下集群民众 | 人口增长强，个体弱 |
| 松鼠族 | 森林采集者 | 木材、坚果、树上移动 |
| 花栗鼠族 | 信使 / 轻步兵 | 移动快，物流强 |

### 行为倾向

| 行为 | 倾向 |
|---|---|
| 地下扩张 | 很高 |
| 挖矿 | 很高 |
| 囤粮 | 很高 |
| 主动宣战 | 低 |
| 防御战争 | 高 |
| 正面作战 | 低 |
| 偷袭 | 中高 |
| 迁都地下 | 高 |

## 5.5 獾国

### 定位

獾国是防御、山地、堡垒、重甲与复仇文明。

他们擅长：

- 山地防御
- 城堡
- 重甲兵
- 长期抵抗
- 边境防线
- 复仇战争

### 核心机制：寸土不让

獾国失去领土后会产生长期复仇记忆。

他们可以暂时忍让，但不会忘记。

一旦国力恢复，獾国很可能重新发动复仇战争，夺回失地。

### 亚种

| 亚种 | 定位 | 特性 |
|---|---|---|
| 欧亚獾族 | 标准民众 / 盾兵 | 防御强，稳定 |
| 蜜獾族 | 狂战士 / 先锋 | 士气极高，劣势时更猛 |
| 美洲獾族 | 游击兵 / 地洞守卫 | 伏击、防守强 |
| 猪獾族 | 山地农民 / 后勤 | 山地农业、补给强 |
| 臭鼬獾族 | 特殊守城兵 | 削弱敌军士气 |
| 白獾族 | 长老 / 祭司 / 法官 | 传统、稳定、威望高 |

### 行为倾向

| 行为 | 倾向 |
|---|---|
| 防御建设 | 很高 |
| 复仇战争 | 很高 |
| 山地扩张 | 高 |
| 主动结盟 | 中低 |
| 遵守条约 | 高 |
| 接受羞辱条约 | 很低 |
| 商业发展 | 低 |
| 镇压叛乱 | 高 |

## 6. 种族关系

自然关系只作为初始倾向，不决定最终命运。

| A | B | 初始关系 | 说明 |
|---|---|---:|---|
| 狗国 | 猫猫国 | -10 | 秩序与独立性格不合 |

| 猫猫国 | 獾国 | -5 | 猫嫌獾粗鲁，獾嫌猫矫情 |
| 狐狸国 | 地鼠国 | +5 | 狐狸需要地鼠资源和通道 |
| 狐狸国 | 獾国 | -20 | 獾讨厌狐狸阴谋 |
| 地鼠国 | 獾国 | -15 | 地下资源与矿山冲突 |

## 7. 国家 AI 参数

每个国家有一组隐藏 AI 倾向值。

| 参数 | 含义 |
|---|---|
| aggression | 侵略性 |
| loyalty | 忠诚 / 守约倾向 |
| expansion | 扩张欲 |
| trade | 商业倾向 |
| intrigue | 阴谋倾向 |
| engineering | 工程倾向 |
| tradition | 传统倾向 |
| curiosity | 探索倾向 |
| revenge | 复仇倾向 |
| stability | 稳定倾向 |

### 7.1 五大种族基础参数

| 种族 | 侵略 | 忠诚 | 扩张 | 商业 | 阴谋 | 工程 | 传统 | 探索 | 复仇 | 稳定 |
|---|---:|---:|---:|---:|---:|---:|---:|---:|---:|---:|
| 狗国 | 60 | 85 | 65 | 45 | 30 | 55 | 70 | 60 | 65 | 80 |
| 猫猫国 | 40 | 45 | 45 | 80 | 70 | 50 | 45 | 55 | 50 | 45 |
| 狐狸国 | 35 | 35 | 55 | 85 | 95 | 45 | 40 | 70 | 60 | 55 |
| 地鼠国 | 25 | 70 | 50 | 55 | 45 | 95 | 60 | 45 | 40 | 70 |
| 獾国 | 45 | 75 | 40 | 35 | 25 | 70 | 90 | 35 | 95 | 85 |

### 7.2 决策示例

```text
如果 侵略性高 + 扩张欲高 + 邻国弱小：
    增加宣战概率

如果 阴谋高 + 正面军力弱：
    优先扶持叛乱或间谍行动

如果 工程高 + 山地/矿区多：
    优先建堡垒、矿井、隧道

如果 复仇高 + 曾失去领土：
    增加复仇战争概率

如果 稳定低 + 贵族权力高：
    增加内乱 / 分裂概率
```

## 8. 自动演化系统

### 8.1 每回合 / 每季模拟流程

推荐以“季节”为基础模拟周期。

```text
1. 资源生产
2. 人口消耗
3. 人口增长 / 下降
4. 城市建设
5. 角色变化
6. 内政判断
7. 外交判断
8. 战争判断
9. 事件生成
10. 历史日志记录
```

### 8.2 国家行为判断顺序

```text
1. 是否有生存危机？
   粮食不足、被围攻、灾害、叛乱

2. 是否有内部问题？
   民心低、贵族叛乱、继承危机

3. 是否有外部威胁？
   邻国扩军、敌军靠近、宿敌复兴

4. 是否有扩张机会？
   无主土地、弱小邻国、资源点

5. 是否有发展机会？
   建城、开矿、贸易、修路

6. 是否有阴谋机会？
   扶持叛军、暗杀将领、破坏联盟

7. 是否触发故事事件？
   英雄、联姻、节日、丑闻、奇迹
```

## 10. 战争系统

  ↓

  ↓
战斗结算
  ↓

  ↓
战后影响

### 10.2 战争原因

| 原因 | 示例 |
|---|---|
| 资源争夺 | 矿山、鱼塘、蜂蜜林 |

| 间谍暴露 | 狐狸国阴谋败露 |
| 饥荒 | 为粮食而战 |
| 扩张主义 | 狗国好战国王上台 |

### 10.3 战争结果

| 内部分裂 | 贵族独立 |

| 英雄建国 | 将军或英雄建立新国 |

## 13. 玩家干预系统

### 13.1 干预原则

玩家干预不应该像后台管理工具，而应该是世界内的轻度神话行为。

视觉上建议表现为：

- 森林神龛
- 小祭坛
- 石碑
- 风铃
- 流星
- 动物信使
- 梦境神谕

### 13.2 温和干预

| 干预 | 效果 |
|---|---|
| 丰收祝福 | 增加某地食物产出 |
| 甘霖 | 缓解干旱，提高农业 |
| 促进和谐 | 降低两个国家敌意 |
| 灵感之风 | 提升文化或科技 |
| 迁徙指引 | 引导部分人口移动 |
| 英雄降生 | 生成特殊角色 |

### 13.3 强力干预

| 干预 | 效果 |
|---|---|
| 天降陨石 | 摧毁区域 |
| 王冠诅咒 | 制造继承危机 |
| 巨兽苏醒 | 生成世界 Boss |
| 边界重画 | 改变领土 |
| 神谕战争 | 提升宣战概率 |
| 灾疫 | 降低人口和稳定 |

### 13.4 喜剧干预

| 干预 | 效果 |
|---|---|
| 巨型毛线球 | 猫猫国幸福上升，生产力下降 |
| 骨头雨 | 狗国狂喜，邻国困惑 |
| 蜂蜜洪水 | 獾国幸福上升，局部混乱 |
| 错误王冠 | 随机民众被立为国王 |
| 鸭子外交团 | 强行召开和平会议，可能失败 |

## 15. 视觉设计方向

### 15.1 总体方向

关键词：

- 清新
- 简洁
- 可爱
- 单机感
- 动物感
- 自然感
- 童话地图
- 低 UI 压迫
- 非页游
- 非复杂仪表盘

### 15.2 需要避免

避免：

- 大量矩形面板
- 页游式 HUD
- 过多按钮
- 密集数字
- 复杂边框
- 强烈“后台系统”感
- 满屏功能入口
- 过度装饰导致不游戏
- 像宣传插画而不像可玩的游戏

### 15.3 推荐视觉结构

```text
顶部：轻量状态条
左侧：极简工具条
中央：地图为主角
右侧：小型国家卡片
底部：轻量事件日志
右下：低调神明干预
```

### 15.4 UI 融入世界的方式

| 功能 | 表现方式 |
|---|---|
| 时间 / 季节 | 木牌、小旗、云朵标签 |
| 地形工具 | 小画笔、自然标记、木牌 |
| 国家信息 | 旗帜、公告板、羊皮纸 |
| 事件日志 | 旅行日记、史官笔记 |
| 神明干预 | 神龛、石碑、图腾、流星 |
| 国家边界 | 柔和色块、花丛、溪流、栅栏 |
| 选中状态 | 光晕、风吹草动、小动物围观 |

### 15.5 最终画面目标

画面应该像：

> 一个清新可爱的动物王国箱庭游戏截图。

而不是：

> 一个复杂网页游戏界面。  
> 一个功能堆满的系统后台。  
> 一张繁琐的幻想插画。

## 19. 当前已确认的设计判断

1. 游戏应更偏单机、清新、简洁，而不是页游式界面。
2. 功能不应以大量方框和按钮外露，而应尽量自然融入世界。
3. 画面中心必须是动物文明世界本身，而不是 UI 面板。
4. 自动演化与事件日志是核心乐趣。
5. 五大种族需要明显行为差异。
6. 玩家干预应轻量、有童话感，而不是后台管理感。
7. 视觉上要削弱程序感，加强动物、自然、箱庭、故事书感。
8. MVP 应优先验证“画世界 → 自动跑 → 观察故事 → 轻度干预”的闭环。
