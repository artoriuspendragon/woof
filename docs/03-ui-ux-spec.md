# UI / UX 规范（§20.B）

> 对应 GDD §20.B，落实 §15（视觉方向）与 §16.5（MVP UI）。本文是可交给美术 / 前端的界面规范：布局、各组件、神明干预、视觉风格禁区与交互流程。
> 数据来源见 [`01-prototype-system-breakdown.md`](./01-prototype-system-breakdown.md)；技术承载见 [`02-technical-implementation.md`](./02-technical-implementation.md)（Canvas 画世界，DOM 浮 HUD）。

---

## 0. 设计信条（一句话）

> **地图是主角，UI 是配角，配角要"长在世界里"。**

来自 §19 的已确认判断：单机、清新、简洁；功能不靠满屏方框和按钮外露；削弱程序感，加强动物 / 自然 / 箱庭 / 故事书感。本规范每一条都服务于此。

---

## 1. 总体布局

承袭 GDD §15.3，但**默认收起**多数面板，让中央地图占满视野：

```text
┌────────────────────────────────────────────────────────────┐
│  [木牌] 第 31 年 · 春      [‖] [▶] [▶▶] [▶▶▶]      [☁ 自动演化中] │ ← 顶部：轻量状态条
├──┬──────────────────────────────────────────────────┬──────┤
│画│                                                    │ 国家  │
│笔│                                                    │ 卡片  │
│工│                  中  央  地  图                       │(选中  │
│具│              （Canvas，唯一主角）                     │ 时    │
│条│                                                    │ 滑入) │
│  │                                                    │      │
├──┴──────────────────────────────────────────────────┴──────┤
│ 📖 第31年春 獾国长老会宣布修建新山堡…   〔小旗筛选〕  〔🌟 干预〕 │ ← 底部：日记式日志 + 右下干预
└────────────────────────────────────────────────────────────┘
```

- **顶部**：木牌时间 / 季节，暂停·倍速，自动演化状态（云朵标签）。
- **左侧**：极简画笔工具条（**仅设计阶段展开**；演化阶段收为一个小图标）。
- **中央**：地图，永远是视觉重心，无遮挡。
- **右侧**：国家卡片，**默认隐藏**，点国家时柔和滑入；点空白滑出。
- **底部**：单行日记式日志（最新一条），可上拉展开完整编年史。
- **右下**：低调的神明干预入口（一颗星 / 一座小神龛图标）。

**两种模式**切换同一画布：
- **设计模式**：左侧画笔工具条展开，右侧/底部最简，引导"画世界 → 放国家"。
- **演化模式**：画笔收起，状态条 + 日志 + 干预登场，引导"观察故事"。

## 2. 顶部状态条

| 元素 | 表现 | 交互 |
|---|---|---|
| 时间 / 季节 | 一块**小木牌**写"第 N 年 · 春/夏/秋/冬"，季节配色微变（春嫩绿、夏浓绿、秋橙、冬白蓝） | 只读；hover 显示精确 tick |
| 暂停 / 倍速 | 四枚**圆形木纽扣**：‖ / ▶ / ▶▶ / ▶▶▶（暂停·1·2·4 倍） | 单选高亮；空格=暂停切换，`1/2/3`=倍速 |
| 演化状态 | 一朵**云朵标签**"自动演化中 / 已暂停" | 只读，呼吸式轻动效 |

样式：无硬边矩形面板，元素直接浮在地图上方，半透明圆角、轻投影、手绘描边。

## 3. 左侧画笔工具条（设计阶段）

§16.5 要求"左侧 4 个绘制工具"。MVP 四支画笔，做成**小画笔 / 自然标记**而非按钮（§15.4）：

| 画笔 | 图标隐喻 | 作用 | 子选项 |
|---|---|---|---|
| 🖌 地形 | 蜡笔 | 刷地表 | 平原/森林/山地/河流/湖泊/沼泽/沙地/雪地 |
| 〰 河流 | 蓝色细笔 | 画河（MVP 作地形/天然边界） | 笔刷粗细 |
| 🚩 国家 | 小旗 | 放国家（选种族）+ 放首都 | 狗/猫/狐/鼹/獾 五旗 |
| ✦ 资源 | 闪光 | 点资源点 | 食物/木/石/铁/鱼/骨/蜜/莓/菇/猫薄荷/闪闪石 |

交互：选画笔 → 子选项以**圆形小调色盘**就近弹出（不占满屏）；左键涂抹、右键擦除；笔刷大小用滚轮。完成后点"启动演化"（一枚发光的"播放"种子按钮），工具条优雅收起。

## 4. 右侧国家卡片

§16.5 必做。**默认不存在**，点击国家时从右侧柔和滑入，呈**旗帜 + 公告板 + 羊皮纸**质感（§15.4），而非数据仪表盘。

```text
   ╭───────────────────────────╮
   │  🐱 猫猫国          [布偶旗] │  ← 旗帜 + 国名 + 种族色
   │  「软爪城」· 喵可大帝 在位   │  ← 首都 + 当前国王
   ├───────────────────────────┤
   │  人口 ▓▓▓▓▓▓░░  312          │  ← 指标用"小木条"而非数字表
   │  食物 ▓▓▓▓░░░░  富足         │  ← 多用"富足/紧张"质感词 + 条
   │  军力 ▓▓▓░░░░░               │
   │  民心 ▓▓▓▓▓▓░░  安定         │
   │  财富 ▓▓▓▓▓▓▓░  繁荣         │
   ├───────────────────────────┤
   │  心情：正忙着办樱花祭 🌸      │  ← 把 AI 当前 goal 翻成"人话"
   │  邻里：与狐狸国…客气又提防    │  ← 关系翻成口语，不露 -20 数值
   ├───────────────────────────┤
   │  〔翻到这国的故事〕          │  ← 跳转日志并按该国筛选
   ╰───────────────────────────╯
```

设计要点：
- 指标用**小木条 / 进度芽**可视化，辅以质感词（富足 / 紧张 / 安定 / 动荡），**避免裸数字罗列**（§15.2 禁区）。
- 隐藏 AI 参数（aggression 等）**不直接示人**；把当前 `goal` 与关系翻译成童话口吻（"正忙着办节日""客气又提防"）。
- 卡片宽度克制（≈ 屏宽 22%），点地图空白即滑出，绝不常驻遮挡。
- 选中国家时，地图上该国领土轻微**光晕 + 小动物围观**（§15.4 选中状态）。

## 5. 底部事件日志 / 编年史

§14 的核心系统在 UI 上呈现为**旅行日记 / 史官笔记**（§15.4），是玩家观察世界的主窗口。

**收起态（默认）**：底部单行，滚动显示最新一条，带等级配色小图标。
**展开态（上拉 / 点击）**：半透明羊皮纸长卷，倒序时间线。

| 日志等级 | 视觉 | 行为 |
|---|---|---|
| 小事件 minor | 浅灰小字 · 🌾 | 一闪而过，可滚回 |
| 中事件 medium | 常规 · 🍃 | 停留稍久 |
| 大事件 major | 加重 · ⚔️ | 地图上对应地点弹小标记 + 轻镜头提示 |
| 史诗 epic | 烫金大字 · ✨ | 短暂"翻页"特写动效，可点击定位 |

筛选（小旗形开关，非下拉框）：按**国家**、按**等级**、按 **tag**（战争 / 外交 / 经济 / 故事）。点任一条目 → 地图镜头平移到事发地（落实 §2.5 可追溯）。文案模板见 GDD §14.3 / §17.3。

## 6. 神明干预（右下）

§13 + §16.5"3 个干预按钮"。表现为**世界内的轻度神话行为**，不是后台工具（§13.1）。入口是右下一座**小神龛 / 一颗低悬星**，点开浮出 ≤3 枚干预（MVP 取温和档 §13.2）：

| MVP 干预 | 图标隐喻 | 效果（接 01 文档干预系统） | 表现 |
|---|---|---|---|
| 丰收祝福 | 麦穗光环 | 选一片地，未来数季食物产出↑ | 落点撒金色光点，麦浪起伏 |
| 促进和谐 | 风铃 | 选两国，降低彼此敌意 | 两都之间飘起风铃与花瓣 |
| 英雄降生 | 流星 | 在某国生成一名特殊角色 | 流星划过 → 该城升起一颗星 |

交互：点干预 → 进入"落点选择"（光标变神龛笔触）→ 地图高亮可选目标 → 确认 → 播放童话动效 + 写一条 `major` 日志。MVP 可设**冷却 / 次数**避免滥用（保持"轻度"§2.2）。强力 / 喜剧干预（§13.3/§13.4，如毛线球、骨头雨、鸭子外交团）作为后续扩展位预留。

## 7. 地图本体的视觉语言（§15.4）

UI 尽量"长在世界里"——很多"界面"其实画在 Canvas 上：

| 信息 | 世界内表现 |
|---|---|
| 国家边界 | **柔和色块 + 手绘描边**，点缀花丛 / 溪流 / 栅栏，绝非生硬格线 |
| 当前选中国 | 领土光晕、风吹草动、小动物围观 |
| 正在发生的事件 | 事发地冒出小图标（⚔️ 火苗 / 🤝 握手 / 🌾 麦束） |
| 主要道路 / 河流 | 浅色曲线，自然蜿蜒 |
| 重要资源点 | 可爱图标（骨头、蜂蜜罐、闪闪石），克制密度 |
| 首都 / 城市 | 小巧建筑簇，随发展度长大 |

§4.4 可读性铁律：**同屏最多强调 5 类**（国界 / 选中国 / 进行中事件 / 主干河路 / 重要资源），其余弱化，杜绝图层堆叠的页游感。

## 8. 视觉风格禁区（§15.2，硬性）

以下一律**不允许**出现，等同验收红线：

- ❌ 大量矩形面板 / 硬边框 / 复杂边框
- ❌ 页游式 HUD、满屏功能入口、密集按钮
- ❌ 密集裸数字、表格化数据墙
- ❌ 强烈"后台系统 / 仪表盘"感
- ❌ 过度装饰致"像宣传插画而不像可玩游戏"
- ❌ 常驻遮挡地图的面板

**正面目标**（§15.5）：截图看起来像"一个清新可爱的动物王国箱庭游戏"，而非"一个复杂网页游戏界面 / 功能堆满的系统后台 / 繁琐的幻想插画"。

## 9. 配色与质感方向

| 维度 | 方向 |
|---|---|
| 主色 | 低饱和、清新自然：草绿 / 天蓝 / 奶白 / 暖木色 |
| 国家色 | 五国各一柔和主色（狗-暖橙、猫-玫粉、狐-赤红、鼹-土棕、獾-灰蓝），用于边界色块与旗帜 |
| 质感 | 手绘描边、纸张 / 木牌 / 羊皮纸纹理、轻投影、圆角 |
| 字体 | 圆润无硬棱的中文字体；史诗日志可用书法 / 烫金点缀 |
| 动效 | 轻、柔、有呼吸感；季节切换、事件冒泡、干预特写都"温和而非炫技" |

无障碍：国家色之外用**图案 / 旗帜形状**辅助区分（色盲友好）；关键状态不只靠颜色。

## 10. 交互流程（玩家旅程）

```text
进入 → [设计模式]
  画地形 → 画河 → 插五面国旗(选种族)选首都 → 点资源点
  → 按下发光的"启动演化"种子
      ↓
[演化模式]
  世界自动跑：边界蠕动、城市长大、日志冒泡
  玩家：暂停/倍速观察 · 点国家看卡片 · 上拉读编年史 · 点日志定位
  想介入时：点右下神龛 → 选干预 → 落点 → 看童话动效
      ↓
  继续观察 → 世界进入新阶段（王朝更替/战争/分裂…全在日志里）
```

对齐 §16.1 MVP 闭环：**画世界 → 自动跑 → 观察故事 → 轻度干预**。

## 12. 交付物建议（给美术 / 前端）

1. **图标集**：4 画笔 + 5 国旗帜 + 11 资源 + 4 日志等级 + 3 干预 + 季节标。
2. **配色规范**：五国主色 + 八地形色 + UI 中性色，含色盲校验。
3. **组件态**：状态条 / 国家卡 / 日志收起·展开 / 干预浮层 的 normal·hover·active 三态。
4. **动效清单**：季节切换、事件冒泡、选中光晕、干预特写、史诗翻页。
5. **禁区对照表**：§8 作为评审 checklist，每版 UI 过一遍。
