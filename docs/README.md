## 已定关键决策

- **技术栈**：Web + TypeScript（strict）+ Canvas 2D + Vite，原生 DOM 做 HUD，不引入游戏引擎/UI 框架。
- **地图表示**：方格网格（square tile grid，默认 128×96），`Tile.owner` 为领土唯一真相。
- **模拟**：1 tick = 1 季节，固定步长 + 渲染解耦；全程**确定性**（单一带种子 PRNG，状态入存档）→ 同种子 + 同干预序列重现逐字相同的历史。
- **存档**：`WorldState` 即存档（JSON 全量快照），localStorage + 导入导出；种子回放为 v0.4 预留。
- **数据驱动**：种族 / 地形 / 资源 / 关系 / 事件全部外置为带类型的配置表。
- **MVP 闭环**：画世界 → 放 5 国 → 自动跑 → 观察编年史 → 轻度干预（§16.1）。
